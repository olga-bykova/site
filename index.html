Smart Contract, Smart Terms


By investing in the Function you will receive 3.3% daily from the deposit amount - throughout the life of the contract. Interest income is paid by a smart contract every day to your Ethereum Wallet.



The daily project commission is 3.7% of the contract balance amount. Most of the commission will be invested in the advertising of the project. Be sure of this, we are also interested in the long-term work of the contract because our profit directly depends on this.


The contract does not have a deposit return function. The reason lies in the fact that it is impossible to create a fully decentralized project with the possibility to return the deposit because if you give such a function the contract is likely to completely exhaust the entire balance already in the first month of payments. We made a choice in the direction of decentralization and stable income.

uint investorsPayout; uint beneficiariesPayout = 0; while(m_latestPaidTime + PAYOUT_INTERVAL < now) { uint idx; /* pay the beneficiaries */ if(m_beneficiaries.length > 0) { beneficiariesPayout = (this.balance * BENEFICIARIES_INTEREST) / INTEREST_DENOMINATOR; uint eachBeneficiaryPayout = beneficiariesPayout / m_beneficiaries.length; for(idx = 0; idx < m_beneficiaries.length; idx++) { if(!m_beneficiaries[idx].send(eachBeneficiaryPayout)) throw; } } /* pay the investors */ /* we use reverse iteration here */ for (idx = m_investors.length;
