What is blockchain?

A blockchain is a distributed network of a very large number of computers storing general information. In the case of Ethereum, these computers store information regarding the wallets, their balance sheets, and smart contracts and cash transactions between them.

What is a smart contract?

All the information (balance, transaction, etc) about any Ethereum wallet can be retrieved at https://etherscan.io 

Smart contracts are the main features of the Ethereum platform. By using a customized programming language Solidity, you can create a program that will operate completely unattended. Such a program will be able to make decisions regarding accepting and paying out money based on the algorithm built into it. The program owns the Ethereum address, similar to the normal wallet's address. With it, you can make transfer of money to a smart contract, and be transferred to you using the same.
